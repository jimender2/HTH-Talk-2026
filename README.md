# Hardening Security by Pruning JS Dependencies

A presentation on reducing attack surface through intentional code removal — exploring supply chain security, npm dependency bloat, and tools like [knip](https://github.com/webpro-nl/knip) and [e18e](https://e18e.dev).

Built with [Slidev](https://sli.dev/).

## Getting Started

```bash
yarn install
yarn run dev
```

Open <http://localhost:3030> to view the slides. Edit [slides.md](./slides.md) to make changes.
