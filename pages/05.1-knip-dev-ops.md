# Why Devs & Ops Love knip Too

<div class="grid grid-cols-3 gap-5 mt-6">

<div v-click class="bg-gray-800/60 rounded-xl p-5 border border-gray-700">
  <div class="text-3xl mb-2 text-center">📦</div>
  <h3 class="text-base font-bold text-green-400 mb-2 text-center">Fewer Packages</h3>
  <div class="text-xs text-gray-400 leading-relaxed">
    Remove unused dependencies from <code class="text-green-300">package.json</code>. Less to install, less to update, less to break.
  </div>
</div>

<div v-click class="bg-gray-800/60 rounded-xl p-5 border border-gray-700">
  <div class="text-3xl mb-2 text-center">🧩</div>
  <h3 class="text-base font-bold text-blue-400 mb-2 text-center">Drop Polyfills</h3>
  <div class="text-xs text-gray-400 leading-relaxed">
    Modern browsers support <code class="text-blue-300">fetch</code>, <code class="text-blue-300">Promise</code>, ES modules natively. Stop bundling polyfills you no longer need.
  </div>
</div>

<div v-click class="bg-gray-800/60 rounded-xl p-5 border border-gray-700">
  <div class="text-3xl mb-2 text-center">📉</div>
  <h3 class="text-base font-bold text-purple-400 mb-2 text-center">Smaller Bundles</h3>
  <div class="text-xs text-gray-400 leading-relaxed">
    Dead code eliminated = fewer bytes shipped. Faster load times, lower bounce rates, better Lighthouse scores.
  </div>
</div>

<div v-click class="bg-gray-800/60 rounded-xl p-5 border border-gray-700">
  <div class="text-3xl mb-2 text-center">🌐</div>
  <h3 class="text-base font-bold text-orange-400 mb-2 text-center">Less Traffic</h3>
  <div class="text-xs text-gray-400 leading-relaxed">
    Smaller bundles mean <span class="text-orange-300">lower bandwidth costs</span> for CDN egress. Real savings at scale.
  </div>
</div>

<div v-click class="bg-gray-800/60 rounded-xl p-5 border border-gray-700">
  <div class="text-3xl mb-2 text-center">⚡</div>
  <h3 class="text-base font-bold text-yellow-400 mb-2 text-center">Faster CI</h3>
  <div class="text-xs text-gray-400 leading-relaxed">
    Fewer deps = faster <code class="text-yellow-300">npm install</code>. Shorter build times, quicker feedback loops.
  </div>
</div>

<div v-click class="bg-gray-800/60 rounded-xl p-5 border border-gray-700">
  <div class="text-3xl mb-2 text-center">🔧</div>

  <div class="text-xs text-gray-400 leading-relaxed">

  </div>
</div>
